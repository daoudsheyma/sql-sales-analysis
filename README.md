📊 Analyse des ventes – MonEntreprise

🔹 Description

Projet d’analyse des ventes sur la base de données “MonEntreprise”.
L’objectif est d’exploiter les données commerciales pour générer des insights clés sur les clients et les produits.

⚠️ Le dataset n’est pas fourni pour des raisons de confidentialité, mais toutes les requêtes SQL et captures des résultats sont disponibles.


🎯 Objectifs du projet
Identifier les clients les plus performants 💰
Déterminer les produits les plus rentables 📈
Analyser le chiffre d’affaires par client et par produit
Extraire les clients ayant un chiffre d’affaires supérieur à 1 000 000 € 🏆
Pratiquer les jointures, agrégations et fonctions analytiques avancées en SQL

🛠 Technologies et outils
SQL Server / T-SQL
Requêtes SELECT, WHERE, JOIN, GROUP BY, HAVING, UNION
Captures des résultats pour démontrer l’analyse

🚀 Compétences développées
Manipulation et analyse de données en SQL 🖥️
Modélisation relationnelle et compréhension des bases de données
Logique Data Engineering (pipeline simple : code → exécution → visualisation)
Analyse et visualisation des résultats
