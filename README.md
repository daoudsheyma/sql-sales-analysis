# sql-sales-analysis
📌 SQL Sales Analysis Project

Description :
Projet d’analyse de données de ventes basé sur SQL Server, incluant extraction, transformation et analyse des données clients et produits.

Fonctionnalités :

Filtrage et exploration des données
Analyse du chiffre d’affaires par client et produit
Détection des clients les plus rentables
Jointures entre différentes tables (clients, ventes)

Compétences :

SQL (SELECT, JOIN, GROUP BY, HAVING)
Data analysis
Data manipulation
Data modeling
